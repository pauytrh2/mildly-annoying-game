# A mildly annoying game
A slightly infuriating shooter game

# Installation
ensure you have rust installed on your system
```bash
rustc --version
cargo --version
```

now clone the repo
```bash
git clone https://github.com/pauytrh2/mildly-annying-game.git
cd mildly-annying-game
```

build and run
```bash
cargo run
```
